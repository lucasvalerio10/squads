Instalação

Siga os passos abaixo. Leva menos de 10 minutos.

1

Instalar o Git

Baixe em git-scm.com e instale com as opções padrão. Depois abra o Git Bash para os próximos passos.

Já vem instalado no Mac. Se precisar, execute no Terminal:

xcode-select --install
2

Instalar o Node.js

Acesse nodejs.org, baixe a versão LTS e instale normalmente. Verifique com:

node --version
3

Instalar o Claude Code

Abra o terminal e execute:

npm install -g @anthropic-ai/claude-code
4

Instalar os Squads

Windows: use o Git Bash · Mac: use o Terminal

curl -fsSL https://raw.githubusercontent.com/lucasvalerio10/squads/main/squads/install.sh | bash

O instalador vai fazer perguntas sobre sua empresa e criar o arquivo empresa.yaml automaticamente.

5

Abrir o Claude Code

Abra o terminal em qualquer pasta do seu projeto:

claude
6

Ver os squads disponíveis

Dentro do Claude Code, digite:

/squads

Vai aparecer o menu completo com todos os 20 comandos disponíveis.

7

Usar o primeiro squad

Experimente com o Squad de Gestão:

/squad-gestao:prioridade

Descreva sua semana e o agente gera seu plano com o contexto da sua empresa. ✅
