Instalação

Siga os passos abaixo para colocar seu time no ar

1

Instalar o Git

Windows: Baixe em git-scm.com e instale com opções padrão.
Mac: já vem instalado ou execute no terminal:

xcode-select --install
2

Instalar o Node.js

Baixe em nodejs.org → versão LTS → instale normalmente.

3

Instalar o Claude Code

npm install -g @anthropic-ai/claude-code
4

Instalar os Squads

Windows: Abra o Git Bash.
Mac: Abra o Terminal. Execute:

curl -fsSL https://raw.githubusercontent.com/lucasvalerio10/squads/main/squads/install.sh | bash
5

Abrir o Claude Code

cd ~/Desktop && claude
6

Ver os squads disponíveis

/squads
