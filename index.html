Perguntas frequentes

Ficou alguma dúvida?

Preciso saber programar pra usar o Squads? +
Não. A instalação é um comando de copiar e colar no terminal. Tem um vídeo explicando cada passo. Se você já abriu o terminal pelo menos uma vez na vida, consegue instalar.
O que é o Claude Code? Preciso pagar por isso? +
Claude Code é a ferramenta da Anthropic onde os agentes rodam. Você vai precisar de uma conta na Anthropic com créditos de API. O custo médio de uso real fica em torno de R$50 a R$100 por mês, dependendo de quanto você usa. Não tem mensalidade fixa, você paga só pelo que usar.
Funciona no Windows e no Mac? +
Funciona nos dois. O Claude Code roda em qualquer sistema operacional. Se tiver dúvida na instalação, os vídeos cobrem Windows e Mac separadamente.
Como recebo o produto depois da compra? +
Na hora. A Kiwify manda o acesso no seu e-mail assim que o pagamento é confirmado. Lá dentro tem o link da área de membros com os vídeos, o script de instalação e o arquivo de configuração.
E se eu travar na instalação? +
Você tem acesso a um grupo exclusivo com outros membros e acesso direto pra me contatar. Quase todo mundo instala em menos de 10 minutos, mas se travar, tem suporte.
